Greenhouse Gas Giants

All the visuals shown were generated using this dataset from Kaggle. The original dataset on Kaggle has 16 columns. The plots below visualize 5 columns from the dataset. The x-axis of all the plots is the column "year" and the y-axis is the emissions produced measured in Millions of tonnes of carbon dioxide equivalent (MtCO2e). There are four types of emission categories: fugitive methane, own fuel use, venting, and flaring. The data visualized is for the parent entity Chevron, emissions produces for it's reporting entity Chevron (each parent entity can have multiple reporting entities, which are entities acquired by the parent entity), and natural gas commodity (there are different types but the plots only show natural gas). In other words, the subset of the data is all the rows that satisfiy the following conditions: parent entity = Chevron, reporting entity = Chevron, and commodity = natural gas.

The basic plot shows the emissions produced from each category per year. The sum of all emissions is the total operational emissions made each year. The line plot shows an increase in emissions for each category. Fugitive methane has the highest rate of increase and is consistently more than all the other categories. Own fuel use and venting have similar rate of change, but venting emissions being consistently more. Flaring emissions make up the smallest amount of emissions, there is a small upward trend as well.

To be able to see the change more clearly, specially for the emission types that make up a small amount, the plot above uses a log scale for the Y-axis. We can see the trend with flaring emissions more clearly, and we see that it follows the same general trend as the other emission types but at a smaller rate. The first plot does not show the exact changes in the flaring emissions because it is very small compared to the other types, using a log scale allows us to see the trend better.

The plot above is a stacked area plot. It allows us to see the total operational emissions produced each year. In the previous plots we were only able to see the emissions of each type individually and the plot above provides more information about emissions produced annually.

The plot above is a straight line approximation of the data. We can see an approximation of the rate at which the emissions produced for each change annually. Fugitive methane emissions change at the fastest rate which is reflected in the plot, the line for that category is the steepest. Flaring emissions change at the smallest rate, line is almost parallel to the x-axis.
